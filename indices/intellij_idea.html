INSTALACIÓN IntelliJ IDEA

Paso 1
Iremos a la pagina web oficial de IntelliJ IDEA

Paso 2
Descargaremos el .exe

Paso 3
Buscaremos la descarga.

Paso 4
Ejecutamos la descarga.

Paso 5
Seleccionamos "next".

Paso 6
Seleccionamos donde queremos que se instale.

Paso 7
Te da varias opciones como crear un icono en el escritorio, crear asociaciones como a .java, .groovy etc

Paso 8
Esperamos a que termine la instalación

Paso 10
Una vez terminada la instalación ejecutamos IntelliJ IDEA.

Paso 11
Nos pedira que aceptemos los términos.

Paso 12
Te da la opción de una prueba gratis o usar una licencia.
Si seleccionas la prueba gratis despues puedes volver a activarte una licencia.

Paso 13
Nos permite exportar los archivos de Visual Studio u omitirlo.

Paso 14
En este caso selecionamos que se importarán y podemos apreciar que aparecen los proyectos.

Paso 15
Podremos ver nuestros proyectos, crear proyectos y clonar repositorios.

Archivo JAVA
Si queremos crear un nuevo proyecto nos da todas las opciones para crear el tipo de archivos.

Archivo JAVA
Vemos como se nos crea un archivo Java.

Archivo HTML
Os enseñamos como crear un html.

Archivo HTML
Vemos como se crea el index.html con la carpeta de css, img y java.
